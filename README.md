# Task-Management-Application
Task Management Application
Overview

A simple Task Management Web Application developed using HTML, CSS, and JavaScript.

The application allows users to:

Add tasks
Mark tasks as completed
Delete tasks
Manage daily activities
Features
Create new tasks
Update task status
Delete tasks
Responsive user interface
Dynamic task handling using JavaScript
